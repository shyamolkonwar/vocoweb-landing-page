3. Implementation Guide: The "Fake Door" Logic
This is the most critical part of the VocoWeb MVP. The design must support the intent tracking mechanism.

Mechanism A: The "Builder" Button ($39)
Visual Cue: Outline button. Looks "safe."
Interaction:
On Click -> Open Modal.
Modal Design: Minimalist. "Join the Waitlist."
Micro-copy: "We are currently onboarding batch #5. Secure your spot."
Metric: Count this as "Low Intent Lead."
Mechanism B: The "Founder" Button ($79)
Visual Cue: Solid White button on Black card. Looks "committed."
Interaction:
On Click -> Redirect to a new page (or Stripe Checkout).
The Page: A "Business Registration" form. Ask for "Company Name" and "Business Type."
Logic: Even if you don't actually process the $79 yet, the act of clicking a button that implies payment and filling out business details proves High Intent.
Metric: Count this as "Validated Customer."
4. Typography & Grid Rules (Summary)
Grid: All sections must align to a 12-column grid.
Spacing: Use py-24 (96px) for section padding.
Borders: Use border-b for every section to create the "stacked paper" look.
Icons: Use Lucide React or Phosphor Icons. Set stroke width to 1.5px (Thin/Technical).
